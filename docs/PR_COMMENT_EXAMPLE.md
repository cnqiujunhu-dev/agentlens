# PR Comment Example

Use this page when you need a copyable example of the Markdown body AgentLens writes for pull request review workflows.

The marker lets a GitHub Actions step update the same comment on every run instead of posting duplicates.

```md
<!-- agentlens-ci-comment -->
## AgentLens CI: FAIL

**Runs:** `.agentlens/review/runs`
**Total:** 2 | **Passed:** 1 | **Failed:** 1
**Scan:** enabled, fail on high

**Artifacts:**
- Run bundle: https://github.com/acme/app/actions/runs/123/artifacts/456
- SARIF: https://github.com/acme/app/security/code-scanning

Found 1 failing trace.

| Trace | Failures |
| --- | --- |
| `.agentlens/review/runs/candidate.json` | no-errors: Found 2 errors; no-send: Forbidden tool called: email.send; final-answer-has-citation: Final response has 0 citation(s), expected at least 1 |

### Trace Diff

**Workflow Signal**

| Signal | Baseline | Candidate | Delta |
| --- | ---: | ---: | ---: |
| Chain events | 2 | 2 | 0 |
| Task events | 2 | 0 | -2 |
| Workflow errors | 0 | 2 | +2 |

**Diff regressions**

- status changed from passed to failed
- task events decreased by 2
- workflow errors increased by 2

<details>
<summary>All trace results</summary>

| Trace | Status | Failures |
| --- | --- | --- |
| `.agentlens/review/runs/baseline.json` | PASS | none |
| `.agentlens/review/runs/candidate.json` | FAIL | no-errors: Found 2 errors; no-send: Forbidden tool called: email.send; final-answer-has-citation: Final response has 0 citation(s), expected at least 1 |

</details>
```

## Generate It

For a directory of traces:

```bash
agentlens ci --runs .agentlens/runs \
  --config evals/default.json \
  --scan \
  --pr-comment-md .agentlens/reports/pr-comment.md \
  --artifact-url https://github.com/acme/app/actions/runs/123/artifacts/456 \
  --sarif-url https://github.com/acme/app/security/code-scanning
```

For a before/after review pack:

```bash
agentlens review .agentlens/runs/baseline.json .agentlens/runs/candidate.json \
  --config evals/default.json \
  --out .agentlens/review \
  --artifact-url https://github.com/acme/app/actions/runs/123/artifacts/456 \
  --sarif-url https://github.com/acme/app/security/code-scanning
```

Then publish `.agentlens/review/reports/pr-comment.md` from a GitHub Actions step. The scaffold generated by `agentlens init --review` includes a marker-based upsert workflow.
